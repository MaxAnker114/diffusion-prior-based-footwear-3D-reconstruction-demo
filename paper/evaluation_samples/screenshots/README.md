## Generation

These assets were generated by:

```bash
source /home/anker/miniforge3/etc/profile.d/conda.sh
conda activate hunyuan3d
cd /mnt/d/Final_Project
python paper/evaluation_samples/generate_phase8b_assets.py
```

The mesh previews are offline orthographic renderings generated from GLB geometry with `trimesh`, `numpy`, and `PIL`. They are intended for thesis figures and quick comparison, not as photorealistic rendering.

## Recommended Use

- Use `phase8b_comparison_grid.png` for the demo/evaluation overview.
- Use `phase8b_metrics_table.png` for the quantitative comparison figure.
- Use `phase8b_s1_sample_sheet.png` as the main method result figure.
- Use S3/S4 sample sheets to discuss baseline differences.
